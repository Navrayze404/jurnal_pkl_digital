URL Google Apps Script tidak boleh kosong.

Format URL wajib: gunakan URL Web App yang berakhiran /exec, contoh https://script.google.com/macros/s/.../exec. Akses kamera membutuhkan HTTPS.

Privasi: foto dikirim langsung dari perangkat Anda ke Google Drive akun Google yang membuat Apps Script. Website ini tidak menyimpan foto di server pembuat aplikasi.

Panduan setup untuk pengguna baru

Backend dibuat di akun Google Anda sendiri, sehingga foto masuk langsung ke Google Drive Anda.

  1. 1
    Buka script.google.com lalu login dengan akun Google Anda.
  2. 2
    Buat project baru.
  3. 3
    Buka repo backend: Code.gs di GitHub, lalu salin seluruh kodenya.
  4. 4
    Paste kode tersebut ke file Code.gs di Apps Script, lalu simpan.
  5. 5
    Klik Deploy → New Deployment, lalu pilih type Web App.
  6. 6
    Atur Execute as: Me dan Who has access: Anyone. Setting ini diperlukan agar frontend bisa mengirim foto ke Drive Anda.
  7. 7
    Klik Deploy, izinkan akses Google Drive jika diminta, lalu copy URL Web App.
  8. 8
    Pastikan URL yang dicopy adalah URL /exec, bukan URL /dev atau halaman project.
  9. 9
    Paste URL https://script.google.com/macros/s/.../exec ke kolom input aplikasi ini.

Catatan privasi: aplikasi frontend hanya mengirim foto ke URL Apps Script yang Anda masukkan. Foto tersimpan di Google Drive akun pembuat Apps Script, bukan di server pembuat website.
